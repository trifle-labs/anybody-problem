@anybody/physics — playable demos

Three browser-playable games built on the generalized physics library. All physics is deterministic fixed-point integer arithmetic — exactly what the matching circom circuit will verify.

about the physics

Each demo loads its preset from physics/presets/<game>.js and runs a deterministic BigInt physics engine (demos/shared/engine.js). The engine handles every object kind, force, boundary, input, and collision response declared in the schema. No floats, ever — every value is an integer scaled by precision.scalingFactor.

about the proofs

The "generate proof" button on each demo expects a compiled circom circuit at public/<game>.wasm + public/<game>_final.zkey. Source circuits are in circuits/games/<game>.circom. Compile them with:

yarn circom flappy
yarn circom dino
yarn circom platformer

(The button will tell you which file is missing if you skip a step.)
